Guide to Import Data

1. View the appropriate template by clicking on the link in the particular category.

2. Right-click on the content and select 'Save As' to download it to the computer.

3. Open the template with spreadsheet application.

4. Edit the content accordingly to add student/lecturer profiles, subject details or student enrollments.

5. Save the file and upload it using 'Browse' ('Choose file' in Mac) button below.

6. Click on 'Import Data' button to submit the upload.
